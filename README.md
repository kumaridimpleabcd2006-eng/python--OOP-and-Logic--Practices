# Python and Logic Practices

A collection of Python programs demonstrating Object-Oriented Programming (OOP) concepts, logic, and system interactions.

## Projects Included:
1. *Python Calculator*: A simple calculator performing basic arithmetic operations.
2. *Smartphone Simulator and Smart Assistant*: An OOP-based simulation that creates a Smartphone class with properties like brand and storage, and includes a realistic ringing sound system (winsound.Beep).
3. *Repeat Name Program *: A simple Python function that repeat a give text multiple time using function parameter and return values.
4. Contact Book *: A simple python program that store and displays contact name and phone number using dictionaries and loops.
